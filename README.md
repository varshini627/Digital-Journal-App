# Digital Journal App ✍️

A simple full-stack Digital Journal Application built as an internship project.

## Features

- Create new journal entries
- View all journal entries
- Edit existing entries
- Delete journal entries
- Search journals by title
- Store data using JSON file storage
- Simple and clean user interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## Project Structure

Digital-Journal-App

├── public
│   ├── index.html
│   ├── style.css
│   └── script.js
│
├── data
│   └── journals.json
│
└── server.js


## How to Run

1. Clone the repository

2. Install dependencies

npm install


3. Start the server

node server.js


4. Open in browser

http://localhost:3000


## About Project

This project demonstrates basic full-stack development concepts including frontend design, backend API creation, data handling, and CRUD operations.
